# Student Result Management System (SRMS)

**Prepared By:** Mirabeth Cyril-Urude  
**Matric Number:** 24/15531

A simple web-based application for managing and calculating student results using HTML, CSS, and JavaScript.

## Features
- Add student results
- Automatic grade calculation
- Clean and simple interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open `index.html` in a browser.
2. Enter student name, matric number, test score, and exam score.
3. Click "Add Result".
4. The total score and grade will be displayed in the table below.
